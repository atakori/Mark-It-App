Mark It

A place for students and choreographers to store and view daily class videos

What is it?

Mark it is an app that allows users to easily upload and view daily dance classes. Choreographers can create classes and upload videos to them to keep them in one easy to access site. On the flip side, dancers can add these classes to their accounts to make sure they never miss a class video!
